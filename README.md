# client-server-nippelboard
* use java 8 
* build with mvn clean compile

## There are also two pre compiled jared versions

### java -jar nippelboard.jar
**Arguments (default)**
* serverPort (8888)
* serverIp (localhost)

### java -jar soundServer.jar 
**Arguments (default)**
* serverPort (8888)
* soundFolder (sounds)
